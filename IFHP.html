Premium Add-On Codes
Premium codes (e.g. E078, E082, E083) boost the fee of a base visit code on the same date of service. Each visit gets its own premium row. Amount = Base Fee × Premium %.
⚠️ When a premium code appears in the export, the app finds the base visit row with the same date, calculates the premium amount automatically, and fills that row's amount field.
